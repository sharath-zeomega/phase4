Angular application on AWS EC2 instance using Docker


1 . First we have to create angular app

2 . After that we have to compile and run the angular app using ng serve command
       Then we have to give ng build then dest folder will be created
3 . We have to dockerise the angular app then we have to create the docker image and then we have to push the files

4 . Then we have to signup for the aws ec2 then we have to create the new instances then we have open instance and
